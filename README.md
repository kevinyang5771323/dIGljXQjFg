# Python图书馆管理系统简介

## 核心功能
- 注册登录
- 角色权限区分
- 个人信息维护
- 图书信息增删改查
- 封面图片上传
- 多条件搜索
- 批量导入导出
- 在线借书
- 预约排队
- 管理员还书
- 状态自动更新
- 近7天借阅折线图
- 分类占比饼图
- 随机好书推荐
- 404统一提示

## 适用场景
- 学校图书馆
- 企业内部图书馆
- 公共图书馆
- 个人图书馆管理

## 技术特点
- 基于 Flask 框架，使用 MySQL 数据库
- 前端采用 HTML, CSS, JS 技术
- 系统界面友好，易于操作
